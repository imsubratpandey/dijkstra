# Dijkstra

Shortest path finder with Dijkstra Algorithm

## Deployment

To deploy this project

Open terminal and navigate to dijkstra

```bash
  cd dijkstra
```

Install packages

```bash
  npm install
```

Start the App

```bash
  npm start
```
